# Шпаргалка по работе с Cursor и агентами

## Быстрые команды Cursor

- **Cmd/Ctrl + K** - Быстрый запрос к Claude
- **Cmd/Ctrl + L** - Чат с контекстом открытых файлов
- **Cmd/Ctrl + I** - Inline редактирование
- **@filename** - Упоминание файла в контексте
- **@agent_go** - Обращение к конкретному агенту

## Шаблоны запросов

### Планирование

```
@agent_go Проанализируй задачу: [описание]
Предложи архитектуру и разбей на подзадачи.
```

### Разработка

```
@agent_go Создай endpoint /api/example
Используй структуру из api/chat.go
Добавь валидацию и обработку ошибок
```

### Тестирование

```
@agent_go Создай unit тесты для api/example.go
Покрой основные сценарии и edge cases
```

### Рефакторинг

```
@agent_go Рефактори api/example.go:
- Улучши обработку ошибок
- Добавь структурированное логирование
- Оптимизируй производительность
```

### Документация

```
@agent_devops Обнови docs/API.md:
- Добавь описание нового endpoint /api/example
- Примеры запросов и ответов
```

## Workflow

1. **Планирование** → Создать issue/план
2. **Разработка** → Feature ветка + работа с агентами
3. **Тестирование** → Автотесты + ручное тестирование
4. **Ревью** → Self-review через агентов
5. **Деплой** → CI/CD или ручной
6. **Мониторинг** → Логи и метрики

## Чеклист перед коммитом

- [ ] Код компилируется
- [ ] Тесты проходят
- [ ] Линтинг пройден
- [ ] Документация обновлена
- [ ] Нет секретов в коде
